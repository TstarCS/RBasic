上次的回家作業

具體的轉換條件如下

  • 高雄市 → 中獎金額 * 2
  • 桃園市 → 中獎金額 * 3
  • 台北市 → 中獎金額 * 5
  • 其他縣市 → 維持原中獎金額
df3 <- df %>%
 mutate(
   實領金額 = case_when(
     grepl('高雄', 購買地區) ~ 中獎金額 * 2,
     grepl('桃園', 購買地區) ~ 中獎金額 * 3,
     grepl('台北', 購買地區) ~ 中獎金額 * 5,
     TRUE ~ 中獎金額))

回家作業： 誰是大賣家?

為了要刺激競爭提升銷售績效，公司決定幫各區區長打考核成績

考核成績計算公式如下：
市場開發度(60%) + 銷售積分(40%)

市場開發度：轄區購買人數/轄區人口數

銷售積分：
- 投注金額 >= 1000 → 3分
- 投注金額 >= 500 → 2分
- 投注金額 >= 50 → 1分

請運用mutate, summarise, case_when, group_by等函數，
幫各區區長計算「市場開發度」、「銷售積分」與「考核成績」等三項分數~
